# Images Folder

Add your app images and icons here:

- `logo.png` - App logo
- `icons/` - Icon folder
- `avatars/` - User avatar images
- `cards/` - Card background images

## Recommended Sizes

- Logo: 512x512px (PNG/SVG)
- Icons: 64x64px (PNG/SVG)
- Avatars: 128x128px (PNG)
- Card backgrounds: 640x400px (PNG)
